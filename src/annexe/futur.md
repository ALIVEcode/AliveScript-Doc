# Le futur d'AliveScript

> Les fonctionnalités en italiques sont loins d'être définitives et nous aimerions entendre votre avis dessus!

## Pour les structures

* Support pour la déclaration de fonctions dans les structures (appelés méthodes)

* Support pour la variable `inst` (signifiant **instance**) afin de faire référence à la structure appelant la
  méthode. (similaire à `this` ou `self` dans d'autres langages)
